# TaskManager
This app manages your daily work.

# Description
This app organizes and manages the work requested by multiple requesters.

# DEMO
You can list tasks for each requester and manage response deadlines and response details.

# Requirement
- ruby '2.6.5'
- rails, '~> 6.0'
- unicorn, '5.4.1
- devise
